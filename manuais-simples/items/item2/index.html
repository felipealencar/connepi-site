Como diretores de modalidade realizam a atribuição de artigos para avaliadores

Voltar

  1. Preliminar:

    Acesse o "Sistema de Gerenciamento de Conferências (OCS)" com sua conta de diretor de modalidade, vá para a página do usuário, e na conferência CONNEPI, clique no link "diretor de modalidade" para exibir os artigos disponíveis.

  2. Etapa 1:

    Nesta página é exibido os artigos submetidos, clicando sobre o título será mostrado os detalhes do artigo selecionado, podendo então, na sessão "Avaliação", inserir novos avaliadores ao artigo através do link "Escolher avaliador".

  3. Etapa 2:

    Aqui é possível inserir como avaliador do artigo; usuários já cadastrado, ou, cadastrar um novo usuário para então seleciona-lo.
    Na primeira opção (cadastrar usuário existente), selecione os usuários desejado e clique em Cadastrar usuários selecionados, se preferir, é possível inserir um usuário por vez, clicando em cadastrar usuário.

  4. Etapa 3:

    Verifique os avaliadores selecionados, em cada usuário, clique em designar para atribuir o artigo ao avaliador.
